Setup Instructions

  1. Import orgchart data: python3 import_orgchart.py REAL-orgchart-export.csv
  2. Individuals provide feedback using the "Individual Feedback" section
  3. Individuals export feedback CSVs (one per manager)
  4. Managers import the CSV files they received
  5. Managers view reports and add their own feedback
  6. Managers export PDF reports for team members
